# 📚 Book Store Website

A simple and responsive multi-page Book Store website built using HTML and CSS. The project includes a home page, authentication pages, about page, and contact page with a clean and user-friendly interface.

## 🌐 Live Demo

🔗 https://ahmedhabash.github.io/BookStore_website/

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/AhmedHabash/BookStore_website.git
```

Navigate to the project folder:

```bash
cd BookStore_website
```

Open `index.html` in your browser.

Or use **Live Server** in Visual Studio Code.
